## 🔧 IF SOMETHING LOOKS WRONG

### Problem: "Nothing changed"
**Fix:** 
- Clear browser cache
- Use incognito window
- Wait 2-3 minutes for Squarespace to update

### Problem: "Some pages still look different"
**Fix:**
- Go to that page in Squarespace editor
- Click Settings → Advanced → Page Header Code Injection
- Delete any `<style>` tags you find
- Save page

### Problem: "Buttons are wrong color"
**Fix:**
- Edit each button block
- Design tab → Button Color → Set to Custom
- Enter: #0066cc
